# 🧠 Design Patterns in C# (.NET Console App)

This repository contains clean, minimal, and practical implementations of classic **Design Patterns** in C# using a .NET Console Application.

Each design pattern is implemented in its own folder with:
- C# source code (`.cs` files)
- A usage example integrated in `Program.cs`
- A `.txt` file that explains the pattern in simple terms

---

## 🧩 Patterns Implemented

Each pattern includes:
- Implementation classes
- `Program.cs` usage example
- `Description.txt` file in plain English

| Pattern Type   | Pattern Name |
|----------------|--------------|
| Creational     | Singleton, Factory, Builder, Prototype, Abstract Factory |
| Structural     | Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy |
| Behavioral     | Observer, Memento, Command, Strategy, State, Mediator, Chain of Responsibility, Visitor, Interpreter, Template Method |

*(more coming with each commit...)*
